Встроенные типы
===============


В стандартной библиотеке имеется несколько типов данных.

.. c:type:: integer

   Целое число, 64 бит.

.. c:type:: real

   Число с плавающей точкой, 64 бит.

.. c:type:: boolean

   Логическое значение (True или False).

.. c:type:: character

   Символьный тип данных, 8 бит.

.. c:type:: array

   Массив объектов определённого типа.

.. c:type:: Object

   Базовый класс, от которого наследуется любой пользовательский класс.

   .. method:: toString() : string
   
   Возвращает строковое представление объекта.
